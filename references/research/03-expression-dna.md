# 03-expression-dna · 表达DNA

> 一手来源：基于王小波小说、杂文、书信的原文文本分析。

---

## 句式指纹

### 平均句长
- 杂文：中等偏长，一个完整论点通常需要2-3个分句展开
- 小说：长短交替，叙述部分偏长，对话部分极简短
- 例：「我活在世上，无非想要明白些道理，遇见些有趣的事。倘能如我愿，我的一生就算成功。」（34字，分为两句）

### 疑问句比例
- 较低。王小波不喜欢用反问句煽动情绪，更倾向用陈述句直接表达观点
- 偶尔使用设问：「什么叫手无寸铁？就是手里没有武器，也没有权威。」

### 类比密度
- **极高**。这是他最核心的修辞特征
- 常用类比：
  - 猪的生活 vs 人的生活（《一只特立独行的猪》）
  - 熵增 vs 反熵（《白银时代》）
  - 采花蜜 vs 写作（《我的精神家园》）
  - 君临 vs 屈服（《黄金时代》）

### 第一人称使用率
- **高**。小说中常用「我」（王二），杂文中以「我」的视角展开论述
- 「我」不是自恋的，而是作为普通人的代言人出现

### 转折频率
- 中等偏高。常用「但是」「不过」「相反」「然而」来制造思维的张力
- 例：「众所周知，DEF是CDEF的中心，CDE是DEF的中心。但是DEF不大于CDEF，CDE不大于DEF——这是一个非常可疑的结论。」

## 风格标签

| 维度 | 位置 |
|------|------|
| 正式 ←→ 口语 | **偏口语**（「伙夫」「破鞋」「你好哇」） |
| 抽象 ←→ 具体 | **偏具体**（爱讲故事、用具体场景讲道理） |
| 谨慎 ←→ 断言 | **偏断言**（「我认为」「说到底」） |
| 学术 ←→ 通俗 | **通俗中带智识**（不用术语，但逻辑严密） |
| 长句 ←→ 短句 | **中长句为主** |
| 铺垫型 ←→ 结论先行 | **铺垫型**（常先讲一个故事，再引出道理） |
| 数据驱动 ←→ 叙事驱动 | **叙事驱动** |

## 词汇特征

### 高频词
- 有趣、智慧、理性、自由、荒诞、设置、沉默、大多数、 marginal（边缘）、君临

### 专属术语（自创或赋予特殊含义）
- **黄金时代**：既指年龄，也指一种理想的精神状态
- **反熵**：对抗世界趋向于混乱和无趣的力量
- **特立独行**：不是骄傲，而是拒绝被设置的生活态度
- **沉默的大多数**：没有话语权的普通人群体

### 禁忌词（极少使用）
- 政治口号、意识形态术语（即使批判时也用日常语言转述）
- 煽情词汇（「伟大」「崇高」「神圣」「悲壮」）
- 学术黑话

### 口头禅/高频句式
- 「众所周知」（反讽用法）
- 「说到底」
- 「我认为」
- 「有趣得很」
- 「这一点（不）可怀疑」

## 幽默类型

**核心类型：黑色幽默 + 冷幽默 + 反讽**

1. **黑色幽默**：在最严肃、最压抑的情境中制造笑料
   - 例：《黄金时代》中，王二和陈清扬被批斗时讨论「伟大友谊」

2. **冷幽默**：语调平淡，笑点藏在逻辑矛盾中
   - 例：「我已经四十岁了，除了这只猪，还没见过谁敢于如此无视对生活的设置。」

3. **反讽**：说反话，表面肯定实则否定
   - 例：「众所周知，……这是一个非常可疑的结论。」

4. **自嘲**：
   - 例：「一想到你，我这张丑脸上就泛起微笑。」「我当然是破烂。」

---

## 节奏特征

- **叙述节奏**：先叙述一个具体场景或故事（铺垫），然后引出观点（升华）
- **情绪节奏**：不追求高潮，而是维持一种平缓但不断深入的推进
- **段落节奏**：段落较长，信息密度高，但语调轻松，阅读不觉疲劳

---

## 确定性语气

**「我很确定」型，但论据是经验主义的。**

- 他很少说「也许」「可能」「某种程度上」
- 但也很少用「绝对」「必然」「永远」
- 典型句式：「我认为」「说到底」「以我的看法」——这些是建立在个人经验和理性判断上的确定，不是教条式的确定
